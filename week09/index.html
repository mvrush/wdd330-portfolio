For example, drawing a rectangle with a stroke and fill is as easy as the following code:
In the HTML page we would use the code:
<canvas id="myCanvas" class="myCanvas" width="200" height="200"> Sorry! Your browser doesn't support Canvas. (If it did there would be a rectangle here). </canvas>

In the JavaScript file we sould then simply use the following code:
var canvas = document.getElementById("myCanvas"); var context = canvas.getContext("2d");
context.strokeStyle = "red";context.fillStyle = "rgba(0, 0, 255, 0.5)";
context.fillRect(10, 10, 100, 100);
context.strokeRect(10, 10, 100, 100);
